Информация о цветовых форматах

OKLCH

Современный перцептуально-однородный цветовой формат, который учитывает особенности человеческого зрения. Состоит из трех компонентов:

  • Lightness (L) - воспринимаемая яркость (0-100%)
  • Chroma (C) - насыщенность цвета (0-0.5+)
  • Hue (H) - оттенок цвета (0-360 градусов)

Более точно отражает восприятие цвета человеком по сравнению с другими моделями.

HSL (Hue, Saturation, Lightness)

Цилиндрическая модель представления цвета, более интуитивная для человека чем RGB:

  • Hue (H) - оттенок цвета (0-360°)
  • Saturation (S) - насыщенность (0-100%)
  • Lightness (L) - светлота (0-100%)

При 0% светлота - черный, при 100% - белый, при 50% - чистый цвет.

HSV (Hue, Saturation, Value)

Альтернатива HSL, также известная как HSB (Hue, Saturation, Brightness). Используется в большинстве графических редакторов:

  • Hue (H) - оттенок цвета (0-360°)
  • Saturation (S) - насыщенность (0-100%)
  • Value (V) - яркость (0-100%)

При 0% значения - черный, при 100% насыщенности и значения - чистый цвет.

RGB (Red, Green, Blue)

Аддитивная цветовая модель, основанная на смешении трех основных цветов:

  • Red (R) - красный компонент (0-255)
  • Green (G) - зеленый компонент (0-255)
  • Blue (B) - синий компонент (0-255)

Используется в мониторах, телевизорах и других устройствах отображения.

HEX (шестнадцатеричный)

Шестнадцатеричное представление RGB цвета в веб-разработке:

  • Формат: #RRGGBB (например: #FF0000 - красный)
  • Сокращенный формат: #RGB (например: #F00 - красный)
  • Каждая пара символов представляет компонент RGB в шестнадцатеричной системе

Стандартный формат для указания цветов в HTML и CSS.

Совет по использованию

Используйте цветовой пикер для интуитивного выбора цвета или вводите значения вручную в любом формате. Все форматы автоматически синхронизируются. Нажмите на кнопку "Копировать" рядом с нужным форматом, чтобы скопировать значение в буфер обмена.
